# Bilibili-sentiment-analysis-and-Irony-detection
Automated Sentiment Analysis and Irony Detection Tool for Bilibili Video Comments
对B站的评论进行讽刺(主要是原神的评论)和情感分析，并在网页中返回结果，仓库包含已经爬取和标注的2400+数据集
